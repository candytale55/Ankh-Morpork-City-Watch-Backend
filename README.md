Este proyecto es crear un backend de cero, manualmente. 

## Paquetes

**Nodemon**: escuchará para que que cuando cambiemos algo en nuestro código y lo guardemos, este recargue el archivo index.js

**Express**: para levantar el servidor y routing (app.METHOD(path,callback))

**Mongoose**: Librería para hacer de intermediario para las consultas de MongoDB

**Dotenv**: Sirve para leer las variables de entorno.

**Insomnia**: (Insomnia.Core-12.6.0) Herramienta de prueba de API que permite crear y enviar peticiones HTTP y observar las respuestas. Simulación de frontend.

**Bcrypt**: Para encriptar las contraseñas. 

**JSON Web Token**: 

Para trabajar con Cloudinary: 

**Cloudinary**:  plataforma en la nube para el almacenamiento, procesamiento y entrega de imágenes y videos. Ofrece una API para subir y gestionar archivos en la nube y se puede integrar con diferentes aplicaciones.

**multer**: Middleware de Node.js para el manejo de archivos enviados a través de un formulario HTML. Permite subir archivos a un servidor o procesarlos en el lado del cliente.

**multer-cloudinary**: es un paquete que permite a Multer almacenar los archivos subidos en diferentes destinos, como el disco local, Amazon S3, Google Cloud Storage, etc.
